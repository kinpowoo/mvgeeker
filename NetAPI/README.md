# NeteaseCloudMusicApi

搜集到以下可用API：

* 搜索API
* 歌曲详情API
* 歌词API

**更新：（增加网易云新版api）**

* 获取歌曲API
* 歌单API
* 歌曲MVAPI

新版API参考

* METO [NeteaseCloudMusicApi](https://github.com/metowolf/NeteaseCloudMusicApi/blob/master/v2/MusicAPI.php)

API返回的是json格式，前端可用js输出。具体使用请看index.php;
